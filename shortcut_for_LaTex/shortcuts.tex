    Open Terminal:                      Ctrl + J

    Create a new file:                  Alt + N

    open next editor tab:               Alt + right arrow

    open previous editor tab:           Alt + left arrow

    view explorer Folder                Alt + 1

    view terminal (toggle panel)        Alt + 4
